Detailed Description

Manages dynamic font sizing based on the dimensions of a JavaFX Scene.

This class automatically adjusts the font size of the Scene's root node when the Scene's dimensions change.

void fr.softsf.sudokufx.utils.DynamicFontSize.initialize ( )
private

Initializes the dynamic font size functionality.

This method sets up listeners for the scene's width and height properties. When the scene's dimensions change, the updateFontSize() method is called to adjust the font size dynamically.

◆ updateFontSize()

void fr.softsf.sudokufx.utils.DynamicFontSize.updateFontSize ( )

Updates the font size based on the current Scene dimensions.

The font size is calculated as 2.19% of the smaller dimension (width or height) of the Scene. This method is called automatically when the Scene's dimensions change.
